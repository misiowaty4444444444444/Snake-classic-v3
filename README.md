# 🇵🇱 PL / Polish

## 🎮 O grze

Snake Classic v3 to klasyczna gra Snake napisana w Pythonie z użyciem `tkinter`.

Gracz steruje wężem, który rośnie po zjedzeniu jedzenia. Gra kończy się, gdy wąż uderzy w ścianę lub w samego siebie.

---

## ✨ Funkcje

- 🐍 Klasyczny Snake (retro styl)
- 🍎 Jedzenie = +1 punkt
- 📈 Wąż rośnie po zjedzeniu
- 💀 Śmierć przy uderzeniu w ścianę
- 💀 Śmierć przy uderzeniu w ciało
- 🔁 Automatyczny restart po Game Over
- ⌨ Sterowanie klawiaturą (WASD + strzałki)

🎮 About the game

Snake Classic v3 is a classic Snake game written in Python using tkinter.

The player controls a snake that grows after eating food. The game ends when the snake hits a wall or itself.

✨ Features
🐍 Classic Snake (retro style)
🍎 Food = +1 point
📈 Snake grows after eating
💀 Death on wall collision
💀 Death on self collision
🔁 Auto restart after Game Over
⌨ Keyboard controls (WASD + arrows)
🎮 Controls
⬆️ Up Arrow / W — move up
⬇️ Down Arrow / S — move down
⬅️ Left Arrow / A — move left
➡️ Right Arrow / D — move right
🚀 How to run
python main.py
